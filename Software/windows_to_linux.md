# Arch Linux Software

Listed here are applications I need to use on this system. Most of these can be found in the CachyOS repository, the Arch Linux repository, or AUR. 

## Communication

### Discord

```shell
paru -S extra/discord
```

### QQ

```shell
paru -S linuxqq
```
